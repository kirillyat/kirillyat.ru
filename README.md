# kirillyat.ru

Личный сайт-визитка / CV. Чистая статика — без сборки и зависимостей.

## Структура

```
index.html      — разметка
css/style.css   — стили (дизайн-токены в :root)
js/data.js      — весь контент (RU/EN) — править здесь
js/main.js      — рендер и интерактив
resources/      — лекции (markdown)
```

## Как править контент

Открой [js/data.js](js/data.js) — там единый объект `CONTENT` с русской и английской версией.
Места, помеченные `TODO`, — заглушки, замени их на реальные данные.

## Локальный запуск

```sh
python3 -m http.server 8000
# открой http://localhost:8000
```

## Деплой

Любой статический хостинг: GitHub Pages, Cloudflare Pages, Vercel, S3 + CDN —
достаточно отдать корень репозитория.

## Фишки

- Настроение: утро в парке — голубое небо, трава, вода, тёплое солнце
- Liquid glass как у Apple: плавающая стеклянная навигация с жидким
  бегунком на пружинной физике, стеклянные карточки с бликами,
  линза-курсор с преломлением (desktop)
- Интерактивное солнце в hero: тяни по дуге — свет всей страницы
  меняется от рассвета до полудня (запоминается в localStorage)
- Бегун 🏃 на прогресс-баре, пыльца в утреннем свете на canvas,
  параллакс-глубина, призрачные номера секций
- Материалы с фильтрами: лекции / семинары / статьи (будущие помечены «скоро»)
- Менторство с формой записи (собирает mailto-письмо, бэкенд не нужен)
- Языки RU/EN/FR — сегментный свитч, выбор сохраняется в localStorage
